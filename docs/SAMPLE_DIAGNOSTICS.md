## Example 1: Check Engine Light - Random Misfire

### Diagnostic Code: P0300
**Random/Multiple Cylinder Misfire Detected**

**Symptoms:**
- Check engine light illuminated
- Rough idle
- Loss of power during acceleration
- Engine hesitation
- Poor fuel economy

**AI Consensus Analysis:**
- **Primary Cause:** Worn or failing spark plugs (85% likelihood)
- **Secondary Cause:** Ignition coil problems (75% likelihood)
- **Additional Factors:** Vacuum leaks, fuel system issues

**Repair Cost Estimate:** $80 - $500
**Estimated Repair Time:** 1-4 hours

**Parts Needed:**
- Spark plugs (set of 4-8): $40-$120
- Ignition coil packs (if needed): $80-$300
- Intake manifold gasket (if needed): $50-$150

**Urgency:** High - Should be addressed within a few days

## Example 2: Catalytic Converter Efficiency

### Diagnostic Code: P0420
**Catalyst System Efficiency Below Threshold (Bank 1)**

**Symptoms:**
- Check engine light on
- Reduced fuel economy
- Failed emissions test
- Possible sulfur smell from exhaust

**AI Consensus Analysis:**
- **Primary Cause:** Degraded catalytic converter (70% likelihood)
- **Secondary Cause:** Oxygen sensor failure (60% likelihood)
- **Additional Factors:** Engine running too rich or lean

**Repair Cost Estimate:** $100 - $2,500
**Estimated Repair Time:** 1-4 hours

**Parts Needed:**
- Downstream oxygen sensor: $100-$300
- Catalytic converter: $400-$2,200
- Exhaust gaskets: $20-$50

**Urgency:** Medium - Can drive but should fix soon for emissions compliance

**Pro Tip:** Always check oxygen sensors first before replacing expensive catalytic converter

## Example 4: ABS System Malfunction

### Diagnostic Code: C1201
**ABS Control Module Malfunction**

**Symptoms:**
- ABS warning light on
- Brake warning light on
- ABS system not engaging
- Possible reduced braking performance

**AI Consensus Analysis:**
- **Primary Cause:** Wheel speed sensor failure (80% likelihood)
- **Secondary Cause:** ABS module internal fault (50% likelihood)
- **Additional Factors:** Damaged wiring, corroded connections

**Repair Cost Estimate:** $100 - $1,500
**Estimated Repair Time:** 1-4 hours

**Parts Needed:**
- ABS wheel speed sensor: $50-$200
- ABS control module (if needed): $400-$1,200
- Wiring harness repair kit: $30-$80

**Urgency:** High - ABS system not functioning properly affects safety

**Safety Note:** Regular brakes still work, but ABS may not engage in emergency stops
